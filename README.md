## Installing

1. Clone repository with "main" branch
2. Install miniconda
3. Run these commands:
   ```bash
   conda env create -f environment.yml
   conda activate Coding_Methods
   ```
4. Check your terminal - it must be like:
   ```bash
	(Coding_Methods) PS C:\Users\rsayf\Desktop\Coding_Methods>
   ```

5. Run this command:
	```bash
	pip install -e .
	```
6. Run tests:
	```bash
	python -u "./src/main.py"
	```
7. Make your scenario with coping from  exisitng in src/hamming/variant15.py and run it same
