Installing

1. Clone repository with "main" branch
2. Install miniconda
3. Run these commands:
conda env create -f environment.yml
conda activate Coding_Methods
4. Check your terminal - it must be like 
(Coding_Methods) PS C:\Users\rsayf\Desktop\Coding_Methods>
5. Run these command:
pip install -e .
6. Run tests
python -u "./src/main.py"
python -u "./src/galua/variant15.py"
